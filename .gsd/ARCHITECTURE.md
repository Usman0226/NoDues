## Data Flow

1. **Request:** Client initiates an action (e.g., submitting a clearance request) via an Axios call.
2. **Middleware:** Request passes through JWT auth validation and Zod schema validation.
3. **Controller:** Business logic processes the request, potentially interacting with multiple models.
4. **Data Layer:** Mongoose performs CRUD operations on MongoDB Atlas.
5. **Side Effects:** Server may trigger background tasks (e.g., sending notification emails via Brevo).
6. **Response:** Server returns a structured JSON response to the client.
7. **Cache Invalidation:** React Query invalidates related tags to ensure the UI stays in sync.
